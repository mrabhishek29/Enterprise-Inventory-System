# Enterprise-Inventory-System

A scalable and maintainable backend application developed using Spring Boot that provides RESTful APIs for managing inventory operations, including product management, user management, and order processing.

The project follows industry-standard practices such as layered architecture, DTO-based communication, exception handling, API documentation, and unit testing to ensure code quality and maintainability.

Features
Product Management (Create, Read, Update, Delete)
User Management (Create, Read, Update, Delete)
Order Management
DTO-based Request/Response Handling
Service and Repository Layer Separation
Global Exception Handling
Input Validation
Swagger/OpenAPI Documentation
Unit Testing with JUnit and Mockito
RESTful API Design
Database Integration using Spring Data JPA
